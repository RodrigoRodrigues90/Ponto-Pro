Como usar o Ponto Pro

🧮 Regras de Cálculo

  • Jornada 6x1: Configurada para a meta de 07:20horas por dia.
  • Líquido: Soma dos dois períodos de trabalho (manhã e tarde), subtraindo o intervalo.
  • Extras: Todo tempo que exceder 7h20min em um único dia é contabilizado na coluna de Extras.
  • Valor Hora: Calculado com base no divisor 220h (Salário / 220).

💾 Gestão de Dados

  • Autossalvamento: O sistema salva seus dados localmente no navegador em tempo real.
  • Exportar: Gera um arquivo .json. usado para gravar o estado atual da sua tabela de horas.
  • Importar: Restaura os dados de um arquivo de backup que você salvou anteriormente.

📄 Relatórios

  • PDF/Imprimir: Gera uma versão limpa da tabela. No celular, selecione "Salvar como PDF" na tela de impressão.
  • Responsividade: No celular, deslize a tabela lateralmente para visualizar as colunas de cálculos.
Dica: Para iniciar um novo mês, exporte o atual como backup e use o botão de "X" nas linhas para limpar a tabela.
